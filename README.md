# 寂灭仙途

一部玄幻修仙长篇小说，计划 1000 章，每章约 3000 字。

## 故事简介

青石镇少年韩默，父母莫名失踪，只留下一枚神秘石珠和半本笔记。

为寻真相，他踏上修仙之路，却发现自己卷入了一场横跨三界的惊天阴谋。

修炼途中，他结识知己，遭遇强敌，探索上古遗迹，揭开仙战真相。

从凡尘到仙界，从蝼蚁到巅峰，这是一段关于成长、牺牲与救赎的传奇。

## 核心设定

- **主角**：韩默（谨慎隐忍，心思缜密）
- **金手指**：神秘石珠（可加速灵药生长，但需付出代价）
- **修炼体系**：炼气→筑基→金丹→元婴→化神
- **世界观**：三界九域，正魔对立，上古仙战之谜

## 创作原则

1. **前期多挖坑**：埋设伏笔，为后期剧情做铺垫
2. **后期填坑**：确保所有伏笔都有合理回收
3. **反复雕琢**：每章完成后反复修改，确保质量
4. **原创性**：借鉴《凡人修仙传》风格，但剧情完全原创

# 《寂灭仙途》第一卷·凡尘问道（1-100 章）详细大纲

## 总体设定

### 修为进度（严格按大纲）
| 章节区间 | 修为境界 | 字数 | 核心事件 |
|----------|----------|------|----------|
| 1-10 | 凡人→炼气 2 层 | 3.2 万 | 获得石珠，加入天剑宗 |
| 11-30 | 炼气 2-3 层 | 6.4 万 | 外门排挤，外门大比 |
| 31-50 | 炼气 4-6 层 | 6.4 万 | 秘境探险，获得机缘 |
| 51-70 | 炼气 7-9 层 | 6.4 万 | 正魔冲突，生死历练 |
| 71-90 | 炼气圆满 | 6.4 万 | 石珠秘密，父母线索 |
| 91-100 | 筑基成功 | 3.2 万 | 突破筑基，第一卷完 |

**总计：100 章，约 32 万字**

### 核心原则（严格遵守）
1. **越阶战斗不可能** - 炼气期永远打不过筑基期
2. **石珠无战力** - 只辅助修炼、感知、保命（小伤）
3. **男主谨慎隐忍** - 不张扬，不炫耀，稳扎稳打
4. **智谋合理** - 用环境、道具、心理战，不是秒杀
5. **身世伏笔后期** - 父母真相 71 章后揭露
